Detailed Description

A simple loopback driver for testing WBC or other controllers. It will set the current position to the reference position, plus configurable white noise.

Subscribed Topics:

  • command (trajectory_msgs/JointTrajectory): Input command. Must contain only joints which are configured in initial_joint_state. However, it may contain less joints

Published Topics:

  • joint_state (sensor_msgs/JointState): The current joint state of all configured joints

Parameters:

  • initial_joint_state (dict): Initial state (position, velocity, effort) of all joints. Syntax is same as in sensor_msgs/JointState
